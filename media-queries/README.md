# Media Queries

## Conceito

As medias queries são recursos que permitem alterar o estilo de uma página de acordo com o dispositivo que está sendo acessado.

---

## Funcionamento

As media queries funcionam definindo condições para aplicar determinados estilos. Quando a condição é verdadeira, os estilos configurados são executados. As alterações mais comuns feitas com media queries envolvem layout, textos, cores e organização dos elementos da página.

---

## Onde é utilizado

Esse recurso é muito utilizado no desenvolvimento de sites responsivos para: adaptar layouts, reorganizar elementos, alterar tamanhos de fontes e imagens... No geral, adaptando os elementos para o dispositivo que está acessadando a página.

---

## Exemplo prático

Sua síntaxe básica é @media e na aplicação ficaria dessa forma:
    
    @media (condição) {

      /* regras CSS */
      
      }
