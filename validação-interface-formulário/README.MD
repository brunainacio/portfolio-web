# Validação de Interface de Formulário

## Conceito

A validação de interface de formulário é o que ocorre quando você insere dados, o aplicativo da Web verifica se os dados estão corretos antes de enviar ao servidor. Isso garante a qualidade dos dados e o funcionamento da aplicação.

---

## Funcionamento

Quando um formulário é preenchido, o sistema verifica cada campo com as regras definidas. Ex: campos obrigatórios, tamanho mínimo/máximo, formato de e-mail/telefone, senhas... Caso alguma informação não bata com os critérios, é retornado uma mensagem indicando o erro e orientando o usuário. Somente após todos os campos atenderem às validações estabelecidas o formulário poderá ser enviado e processado pelo sistema.

---

## Onde é utilizado

A validação de formulários é utilizada em sistemas, sites e aplicações que recebem informações dos usuários por meio de campos de preenchimento. É comum em telas de cadastros, formulários de logins, alterações/recuperações de senha...
