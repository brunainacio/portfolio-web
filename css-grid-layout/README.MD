# CSS Grid Layout

## Conceito

O módulo de layout em grade oferece um sistema de layout em grade, com linhas e colunas e permite que desenvolvedores criem layouts complexos com facilidade. Esse recurso facilita a criação de uma estrutura responsiva sem a necessidade de usar elementos de posicionamento.

---

## Funcionamento

O layout grid é composto por, contêiner-grid que é o elemento principal que organiza os itens e os itens grid, que são os elementos internos posicionados dentro dessa grade. 

Principais funcionalidades: 
* display --> define o comportamento da grade
* grid-template-columns --> define quantidade e tamanho das colunas
* grid-template-rows --> define quantidade e tamanho das linhas
* gap --> espaçamento entre linhas e colunas
* justify-content --> alinha a grade horizontalmente
* align-content --> alinha a grade verticalmente
* justify-items --> alinha itens horizontalmente
* align-items --> alinha itens verticalmente

---

## Onde é utilizado

O CSS Grid Layout é utilizado para criar estrutura como: 
* Cabeçalho
* Rodapé
* Menu lateral
* Conteúdo principal
* Galeria de imagens
* Layout responsivos
* Dashboards
